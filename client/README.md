# React Typescript Boilerplate (without CRA)

## Features:
* React / React Router
* TailwindCSS v3
* Typescript
* Webpack
* Babel
* ESLint
* Sass

## In the project directory, you can run:

### Development:

```Shell
yarn run dev
```

### Start:

```Shell
yarn run start
```

### ESLint format code:

```Shell
yarn run lint:fix
```
